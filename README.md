1. Project Title
Heart Disease Analysis Using Machine Learning
2. Project Description
The Heart Disease Analysis system is a machine learning based application designed to analyze patient health data and predict the likelihood of heart disease. The system uses a dataset containing various medical attributes such as age, cholesterol level, blood pressure and lifestyle indicators.
Machine learning models trained in Python analyze the dataset and generate predictions based on patient data.
3. Dataset Description
The dataset used in this project is Heart_new2.csv. It contains health related parameters that help determine the likelihood of heart disease.
Important attributes include:
Age, Gender, Cholesterol Level, Blood Pressure, Smoking Habits, Physical Activity and Medical History.
These attributes are used as input features for the machine learning model.
4. Technical Architecture
Frontend Layer:
HTML pages are used to provide the user interface.
Pages included:
base.html – Contains graphs
index.html – Main prediction page
story.html – Explains the project and analysis
Backend Layer:
Python is used for data preprocessing, model training and prediction generation.
5. System Workflow
1. Dataset is loaded into Python.
2. Data preprocessing is performed.
3. Machine learning model is trained.
4. User enters health data in the interface.
5. Model predicts the risk of heart disease.
6. Result is displayed to the user.
6. Technology Stack
Programming Language: Python
Machine Learning: Scikit‑learn
Data Processing: Pandas, NumPy
Visualization: Matplotlib
Frontend: HTML
Dataset: Heart_new2.csv
7. Testing
Testing ensures that the system produces correct predictions.
Example Test Cases:
Valid patient data → Prediction result
Missing values → Error message
Invalid input → Validation warning
8. Results
The system successfully analyzes patient health data and predicts the likelihood of heart disease using machine learning techniques. The graphs help visualize demographic, lifestyle and medical patterns in the dataset.
9. Conclusion
This project demonstrates how machine learning can assist in healthcare analysis. By using Python and machine learning libraries, the system predicts heart disease risk and helps understand factors influencing cardiovascular health.
